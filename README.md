
# 🎉 CHU 10K Game

- 這是一款為了慶祝[宅宅軍團長](https://www.youtube.com/channel/UCEky8OgUsk5zYwFlkMpC6ig)的 YouTube 頻道達成 10,000 訂閱所特別製作的小遊戲！
- 總遊戲時長差不多為10分鐘，以放置的方式為主，目標是在遊戲中達成10k訂閱。
